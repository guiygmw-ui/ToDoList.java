# Lista de Tarefas (To-Do List) em Java

Este é um projeto de console desenvolvido em Java que funciona como um gerenciador de tarefas diárias. O foco principal deste projeto foi trabalhar com estruturas de dados estáticas e algoritmos de manipulação de vetores.

## Funcionalidades

- Adicionar Tarefa: Permite cadastrar novas obrigações com um limite máximo de 10 itens na lista.
- Remover Tarefa: Permite deletar uma tarefa específica informando o número correspondente na tela.
- Listar Tarefas: Exibe de forma numerada e organizada todas as atividades ativas no momento.
- Validação de Limites: O sistema avisa quando a lista está cheia e impede a inserção de novos itens até que algum seja removido.

## O que aprendi e apliquei neste projeto

Este código demonstra minha base em lógica de programação e algoritmos essenciais:
- Estruturas de Dados Estáticas (Arrays): Utilização de vetores de tamanho fixo (`String[]`) para armazenar os dados na memória.
- Algoritmo de Deleção e Reorganização: Criação de uma lógica personalizada com estruturas de repetição (`for`) para remover um item do meio do vetor e "arrastar" os elementos seguintes para cobrir o espaço vazio, mantendo a lista ordenada.
- Controle de Fluxo: Uso eficiente de laços de repetição (`while`) e condicionais (`if/else`) para gerenciar as escolhas do usuário no menu do console.

## Como testar o projeto no seu computador

Você só precisa de 3 passos simples para rodar este código:

1. Baixe o arquivo: Baixe o arquivo `Scratch.java` (ou crie um arquivo com esse nome) e salve-o em uma pasta no seu computador.
2. Abra no seu editor: Abra essa pasta no programa que você usa para programar (como o VS Code, IntelliJ ou Eclipse).
3. Dê o Play: Abra o arquivo e clique no botão de "Executar" (Play) do seu editor de código. O gerenciador de tarefas abrirá direto no console!
